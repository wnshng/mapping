# Maping MVP

네이버지도 즐겨찾기 정리를 더 쉽게 하기 위한 리스트 중심 웹앱입니다.

## 포함 기능

- 장소 단건 추가
- 텍스트/CSV 일괄 입력
- 카테고리 자동 분류(규칙 기반) + 수동 보정
- 태그 추천/수정 + 태그 관리(이름 변경/삭제)
- 지역(시/구) 자동 추출
- 메모/방문 상태/저장 이유 관리
- 중복 후보 탐지(정확 일치 + 유사 이름/동일 지역)
- 중복 그룹 병합
- 검색/필터/정렬/뷰 탭
- 대량 선택 일괄 편집
- CSV/JSON 내보내기
- 네이버 로그인 OAuth 진입점 + 반자동 가져오기 북마클릿 헬퍼

## 실행 방법

정적 파일이라 서버 없이도 열 수 있지만, 네이버 OAuth/클립보드는 보통 `http://localhost`에서 더 안정적으로 동작합니다.

```bash
cd "/Users/junhyung/Documents/New project/mapping"
python3 -m http.server 5500
```

브라우저에서 `http://localhost:5500` 접속.

## 네이버 연동 방식

이 MVP는 네이버지도 즐겨찾기 공식 API 제한을 전제로 아래 방식으로 구현되어 있습니다.

1. 네이버 OAuth 로그인(사용자 인증/계정 확인)
2. 네이버 저장목록 페이지에서 북마클릿으로 항목 텍스트 추출
3. 앱의 "텍스트 가져오기"에 붙여넣어 일괄 정리

### OAuth 설정 참고

- 네이버 개발자센터에서 애플리케이션 생성
- `Client ID`, `Redirect URI`를 앱 화면에 입력
- 로컬 테스트 기준 Redirect URI/Callback URL 권장값: `http://localhost:5500/naver-callback.html`
- 로그인 버튼 클릭

> 참고: 브라우저/CORS 조건에 따라 프로필 API 호출이 제한될 수 있습니다. 이 경우에도 반자동 가져오기 흐름은 계속 사용할 수 있습니다.
